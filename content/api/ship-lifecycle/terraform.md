## terraform

A `terraform` step will deploy terraform resources.





### Optional Parameters


- `path` - the directory within `installer` within which to run terraform.


### Examples

```yaml
lifecycle:
  v1:
    - terraform: {}
```

```yaml
lifecycle:
  v1:
    - terraform:
        path: terraform/
```
