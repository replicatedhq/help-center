## generate

**type object**

Adds a step that will generate a support bundle. It has one argument, use_defaults


```yaml
lifecycle:
  v1:
    - generate:
        use_defaults: true
```


### Optional Parameters


- `use_defaults` - Whether to include the default support bundle files - cpu info, docker ps, etc
