## docker.info

Collect info about the Docker daemon


```yaml
collect:
  v1:
    - docker.info:
        output_dir: /docker/daemon/
        timeout_seconds: 10
        description: Info about the docker daemon
```


### Outputs

    
- `docker_info.json` - A pretty-printed JSON representation
