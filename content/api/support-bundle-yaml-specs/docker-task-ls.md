## docker.task-ls

List docker swarm tasks


```yaml
specs:
  - docker.task-ls:
      output_dir: /swarm/tasklist
      Filters:
        name:
          - cooltool-api-
          - cooltool-backend-
```


### Optional Parameters


- `Filters` - Filters for tasks


- `include_empty` - Include empty files generated by this command in the support bundle



### Outputs

    
- `task_ls.json` - JSON output
