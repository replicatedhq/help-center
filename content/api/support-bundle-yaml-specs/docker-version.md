## docker.version

Get the version of the docker server


```yaml
specs:
  - docker.version:
      output_dir: /docker/version
```


### Optional Parameters


- `include_empty` - Include empty files generated by this command in the support bundle



### Outputs

    
- `docker_version.json` - JSON output
