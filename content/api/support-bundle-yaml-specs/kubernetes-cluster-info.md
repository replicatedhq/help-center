## kubernetes.cluster-info

Get the Kubernetes cluster info


```yaml
collect:
  v1:
    - kubernetes.cluster-info:
        output_dir: /kubernetes/
```


### Outputs

    
- `cluster_info.json` - A json file holding the Kubernetes cluster info
