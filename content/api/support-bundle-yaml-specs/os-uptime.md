## os.uptime

**type object**

Collect information about the system's uptime


```yaml
collect:
  v1:
    - os.uptime:
        output_dir: /system/uptime
```


### Optional Parameters


- `template` - Template for the human-readable output



### Outputs

    
- `contents` - The contents of the uptime file
