## when

The `when` property is used to control whether an item will be shown to the end user.




### Examples

```yaml
config:
  v1:
    - when: >-
        '{{repl or (ConfigOptionEquals "select_one" "external")
        (ConfigOptionEquals "select_one" "embedded")}}'
```
